# Ramreiso Portfolio

A responsive personal portfolio built with React, Vite, Three.js, React Three Fiber, and CSS.

## Overview

This site presents selected web development projects, technical skills, open-source contributions, and contact links in a clean single-page portfolio. It includes data-driven project cards, real project previews, an interactive 3D skills section, scroll-triggered section transitions, and reduced-motion support.

## Development

- Install dependencies: `npm install`
- Run locally: `npm run dev`
- Build for production: `npm run build`

## Deployment

This repository deploys to GitHub Pages through GitHub Actions. In the repository settings, set **Pages** → **Build and deployment** → **Source** to **GitHub Actions**, then push to `main`.

## Features

- React + Vite frontend
- Reusable project cards with live demo links and preview images
- Interactive 3D skills visualization using `@react-three/fiber` and `@react-three/drei`
- Responsive layout for desktop and mobile screens
- Accessible profile/contact links
- Reduced-motion handling for animation-sensitive users
