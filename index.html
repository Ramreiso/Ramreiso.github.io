Projects

SMS Spam Detection

Build SMS spam detection models with dense neural network, Long Short Term Memory(LSTM) and Bidirectional-LSTM (BiLSTM)

Accomplishments
  • Tools: Python, Tensorflow, NumPy, Pandas, Matplotlib
  • The data set is summarized and visualized through NumPy, Pandas, WordCloud and Seaborn
  • Downsampling is applied to solve the imbalanced data.
  • Text pre-processing includes Tokenization, Sequencing and Padding.
  • Dense Neural Network, LSTM and BiLSTM models are trained and evaluated.
  • Dense Network model yields the highest validation accuracy with 95%.
  • An input field is created where the user can type message to test whether the message is a spam or ham.
